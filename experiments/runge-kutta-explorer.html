About Runge-Kutta

What are these methods?

Runge-Kutta methods are numerical techniques for solving ordinary differential equations (ODEs) of the form:

dy/dt = f(t, y)

Given an initial value, these methods approximate the solution by taking discrete steps forward in time.

Why different methods?

  • Euler - Simple but inaccurate. Uses only the slope at the current point.
  • RK2 - Checks slope at midpoint for better accuracy.
  • RK4 - Evaluates four slopes with weighted average. Industry standard.

Key tradeoffs

Accuracy vs Speed: Higher-order methods need more computations but can use larger steps.

Stability: Some methods become unstable with large step sizes.

Energy: For physics, maintaining conserved quantities matters.

Current Selection

Method: RK4

Fourth-order Runge-Kutta. Evaluates derivative at four points per step:

k1 = f(t, y) k2 = f(t+h/2, y+h*k1/2) k3 = f(t+h/2, y+h*k2/2) k4 = f(t+h, y+h*k3) y_new = y + h/6*(k1+2k2+2k3+k4)

Error scales as O(h^4). Gold standard for general ODE solving.
